# Mintlify – Documentation Website

A static landing page for **Mintlify**, the intelligent documentation platform. The site presents Mintlify’s product, features, and customer stories in a single-page layout.

## Preview 

![Website Preview](./assets/127.0.0.1_5500_index.html.png)

## Overview

This project is a front-end clone/inspiration of the Mintlify marketing site. It highlights:

- **The Intelligent Knowledge Platform** – Documentation built for both humans and AI
- **Features** – LLMs.txt & MCP, self-updating docs with an agent, and an AI assistant
- **Enterprise** – Partnership, compliance, and customer stories (e.g. Anthropic)
- **Customers** – Case studies and logos (Anthropic, Coinbase, Microsoft, Perplexity, HubSpot, X, PayPal, Lovable, and more)

## Tech Stack

- **HTML5** – Semantic structure and content
- **CSS** – Layout, typography, and responsive styling
- **Assets** – SVG icons/logos, PNG images, MP4 videos

## Sections

| Section | Description |
|--------|-------------|
| Hero | Headline, email CTA, and hero illustration |
| Companies | Logos of customers/partners |
| Features | LLMs.txt & MCP, Agent, Assistant with videos |
| Enterprise | Enterprise value, compliance, Anthropic story |
| Case Studies | Customer carousel (Perplexity, X, Kalshi, etc.) |
| Footer | Links and final CTA |
